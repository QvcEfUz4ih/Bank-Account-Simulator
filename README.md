 # Bank Account Simulator

A Python application that simulates basic banking 
operations with account health monitoring.

## Features
- Enter account holder name
- Enter current balance, deposit and withdrawal amounts
- Automatically calculates new balance
- Account status monitoring:
  - Overdrawn: balance below R0
  - Low Balance: balance below R500
  - Account is Healthy: balance above R500
- Displays full account summary in Rands (R)

## Technologies Used
- Python 3

## How To Run
1. Download the file
2. Open in Python IDLE or any code editor
3. Run the program
4. Enter account details and transaction amounts
5. View your account summary and status
